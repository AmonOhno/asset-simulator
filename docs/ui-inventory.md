# UI 要素インベントリ（ナンバリング）

キャプチャへのナンバリングは 2 カテゴリに分ける：

- **F##（固定値 / 青）** — コードに埋め込まれた静的要素（ナビ・リンク集・フッター等）
- **D##（変数 / 橙）** — バックエンド（Supabase）や状態から取得する動的要素

注釈付きキャプチャ: [`captures/local-app/annotated-mobile-full.png`](captures/local-app/annotated-mobile-full.png)
（`npm run annotate:local` で再生成。セレクター定義は `capture/maps/local-app.json`）

## 固定値（F）

| No. | 要素 | 実装 | 現行サイトの対応要素 |
|-----|------|------|--------------------|
| F01 | サイトロゴ・市名 | `Header.tsx` | ヘッダーロゴ（画像） |
| F02 | スキップリンク（フォーカス時のみ表示） | `App.tsx` | 「本文へスキップ」相当 |
| F03 | 言語切替（I18N） | `LangSwitcher.tsx` | 多言語（機械翻訳）リンク |
| F04 | 文字サイズ・色合い切替（A11Y） | `A11yToolbar.tsx` | 文字サイズ変更・色合い変更 |
| F05 | サイト内検索フォーム | `SearchBar.tsx` | サイト内検索ボックス |
| F06 | グローバルナビ（分野別 6 カテゴリ） | `Sections.tsx` | 分野別タブナビ |
| F07 | ―（欠番） | ― | モバイルはハンバーガーではなく常時表示ナビを採用したため欠番 |
| F08 | カテゴリグリッド（実リンク） | `Sections.tsx` | 分野別メニュー一覧 |
| F09 | 区役所グリッド（10 区・実リンク） | `Sections.tsx` | 各区役所へのリンク |
| F10 | よく利用されるメニュー（実リンク） | `Sections.tsx` | よく見られるページ等 |
| F11 | フッターリンク集 | `Sections.tsx` | フッターナビ |
| F12 | フッター所在地・電話・著作権 | `Sections.tsx` | 所在地・Copyright |
| F13 | ログイン導線（パスキー） | `Header.tsx` | （現行サイトに無し・新規） |

## 変数（D）— バックエンド取得など

| No. | 要素 | データソース | 現行サイトの対応要素 |
|-----|------|-------------|--------------------|
| D01 | 緊急情報バナー | Supabase `emergency_notices`（未接続時フォールバック） | 緊急情報枠 |
| D02 | 新着情報リスト | Supabase `news` | 新着情報（news.html） |
| D03 | イベント情報 | Supabase `events` | イベント・お知らせ枠 |
| D04 | 認証状態表示 | Supabase Auth セッション | （現行サイトに無し・新規） |
| D05 | 検索結果 | クライアント検索（入力に応じ変化） | 検索結果ページ。検索実行後のみ表示されるため注釈対象外 |

## 現行サイト用マッピング

現行サイト（https://www.city.saitama.lg.jp/）向けのセレクター定義は
`capture/maps/current-site.json` にある。**ネットワーク制限により未検証**のため、
アクセス可能な環境で `npm run annotate:current` を実行し、`legend.md` の
未検出（❌）を見ながらセレクターを修正すること（docs/missing-info.md 参照）。
現行サイト固有の変数要素としてバナー枠を `D06` として予約している。
